# دفتر مدير الفرع — تطبيق PWA لمحلات الذهب والفضة

تطبيق محاسبة يعمل **دون اتصال** ومحفوظ بالكامل على جهازك فقط. مبني بـ HTML/CSS/JS خالص بدون أي أدوات بناء.

## المرحلة الحالية (تم إنجازها)

- ✅ **الصفحة الرئيسية** — لوحة تحكم مع مؤشرات اليوم (KPIs)، بلاطات الدفاتر، نسخ احتياطي (تصدير/استيراد JSON)
- ✅ **دفتر اليومية** — قيود تفصيلية كاملة
  - يدعم **الذهب** بالأعيرة `18 / 21 / 22 / 24` والأشكال `مشغول / كسر / صافي`
  - يدعم **الفضة** نقاء `925 / 999` والأشكال `مشغول / كسر / صافي`
  - يدعم **النقد** (قبض / صرف)
  - حساب تلقائي للإجمالي = الوزن × سعر الجرام
  - طرق دفع: نقداً / بنك / آجل
  - عرض **تفصيلي** (جدول قيود) و**إجمالي** (ملخص لكل عيار ونوع)
  - تنقل بين الأيام + زر "اليوم"
  - طباعة
- ✅ **PWA كامل** — manifest + service worker + أيقونات، يثبَّت كتطبيق ويعمل offline

## الدفاتر القادمة (جاهزة في التنقل، مرجأة للمرحلة التالية)

- دفتر الموردين
- دفتر البنك والصندوق
- دفتر المصاريف
- دفتر السلف
- دفتر المخزون

## التشغيل

التطبيق PWA ويجب تقديمه من خادم HTTP (لأن الـ Service Worker لا يعمل مع `file://`).

### الطريقة الأسرع (Python)

```bash
cd gold-ledger
python3 -m http.server 8080
```

ثم افتح: `http://localhost:8080`

### بدائل

- **Node.js:** `npx serve .`
- **VS Code:** امتداد *Live Server*
- **نشر على الإنترنت:** أي استضافة ثابتة تعمل (Netlify, Vercel, GitHub Pages، أو أي استضافة عادية).

> بعد أول زيارة أونلاين، سيعمل التطبيق بالكامل offline.

## تثبيت التطبيق على الهاتف/الكمبيوتر

- **أندرويد/كروم:** اضغط على "تثبيت التطبيق" في شريط العنوان أو من قائمة المتصفح
- **iOS/سفاري:** زر المشاركة ← "إضافة إلى الشاشة الرئيسية"
- **ديسكتوب:** أيقونة التثبيت في شريط العنوان

## البيانات والنسخ الاحتياطي

- جميع البيانات محفوظة في `localStorage` على جهازك فقط. لا يتم إرسالها إلى أي خادم.
- **مهم:** احرص على تصدير نسخة احتياطية بشكل دوري من الصفحة الرئيسية.
- استيراد النسخة الاحتياطية متاح على نفس الجهاز أو على جهاز آخر.

## نموذج البيانات (قيد اليومية)

```js
{
  id: "...",                // معرّف تلقائي
  date: "2026-04-22",       // YYYY-MM-DD
  time: "14:30",            // HH:MM
  flow: "in" | "out",       // وارد أو صادر
  category: "gold" | "silver" | "cash",
  karat: "21" | "24" | ... | "925" | null,  // العيار/النقاء
  form: "worked" | "broken" | "pure" | null, // الشكل
  weight: 25.500,           // الوزن بالجرام
  unit_price: 285.50,       // سعر الجرام بالريال
  total: 7280.25,           // الإجمالي بالريال
  payment_method: "cash" | "bank" | "credit",
  description: "...",       // البيان
  ref: "..."                // رقم الفاتورة/المرجع
}
```

## التمديد (للدفاتر القادمة)

إضافة دفتر جديد تتطلب:
1. إنشاء ملف `js/pages/xxx.js` يُصدِّر دالة `renderXxx(container)`
2. إضافة مفتاح للمخزن في `db.js` إذا احتاج تخزين جديد
3. تسجيل المسار في `routes` داخل `js/app.js`
4. تفعيل الرابط المعطَّل في الشريط الجانبي في `index.html`

الهيكل جاهز وقابل للتوسع مباشرة.
